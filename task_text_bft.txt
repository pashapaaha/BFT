Задание.
Разработать веб приложение по генерации сообщений из текстового шаблона.
Например:
Шаблон 1: Добрый день, ${name}!
Запрос: Из шаблона 1 по параметрам {name: ‘Владимир’}  сгенерировать сообщение
Ответ: ‘Добрый день, Владимир!’

Шаблон 2: Счастливые числа: ${numbers join ‘ ’}
Запрос: Из шаблона 2 по параметрам {numbers: [4,7,9]}  сгенерировать сообщение
Ответ: ‘Счастливые числа: 4 7 9’

Технологии
Framework: spring boot 2
Язык программирования: kotlin
Интерфейс: swаgger для спринга
Сборка: maven
Работа с бд: hibernate или ebean

Спроектировать\реализовать:
1.  Авторизация
2.  Авторизованному пользователю доступны CRUD действия (добавление, чтение, обновление, удаление) с текстовыми шаблонами.
    Шаблон должен включать идентификатор, текст шаблона, перечень  доступных переменных, признак активности шаблона и т.д.
3.  Без авторизации доступно действие по генерации сообщения по идентификатору шаблона и входным параметрам.
    Создавать сообщения можно синхронно и асинхронно (разные методы).
4.  Формат шаблона  должен поддерживать вставку переменных, списков, вложенных переменных. Пример входных параметров:
{
name: ‘Иван’,
company: {
	name: ‘БФТ’,
	branches: [{ title : ‘Пензенский’}, { title : ‘Московский’}]
}
}
